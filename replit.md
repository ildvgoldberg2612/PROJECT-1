# Yagel & Amiad Goldberg — Personal Website

A single-page, static HTML/CSS site celebrating brothers Yagel and Amiad
Goldberg. No build step, no dependencies — just `index.html` and `style.css`.

## Running it
- The `Website` workflow runs `python3 -m http.server 3000` to serve the
  folder for the Replit preview.
- The real, public site is meant to be hosted via GitHub Pages (see
  README.md for the handout instructions this template was based on).

## Structure
- `index.html` — page content and structure.
- `style.css` — bold & colorful styling (gradient headline, blob background
  accents, three highlight cards).

## User preferences
- Visual style: bold & colorful.
- Content: personal site for Yagel and Amiad Goldberg — "Best brothers
  around" — emphasizing they help each other, rarely fight, and love being
  together.
